Змейка — наблюдение

Управление только через API. Здесь только просмотр.

Правила игры

  • Игра: 5 уровней, раунд 3 минуты. Общий счёт — сумма очков по уровням × множитель уровня (1–5). Чем сложнее уровень, тем больше весит результат.
  • Предметы на поле:
    Красное яблоко — +1 очко.
    Золотое — +10 очков.
    Кислотное (зелёное) — бомба: обнуляет счёт, наступать нельзя.
    Голубое — баф скорости (движение дважды за тик на 15 тиков).
    Фиолетовое — щит: до 2 штук, одно столкновение забирает щит вместо жизни.
  • Смерть: врезался в стену, себя или другую змейку. С головы и каждого второго сегмента падают красные яблоки — столько же очков снимается (счёт не ниже 0). Через 5 тиков — респавн в случайной свободной клетке.
  • Змейки: цветные — игроки по API, серые — боты. Управление только через общение с сервером по API.
